# 小市民權益保護99招

### 刑事法律急救手冊

當「權利」遇上「公權力」，是小蝦米對上大鯨魚，還是勢均力敵？

身為老百姓，面對警調機關的權力行使，該如何尋求自保之道？

本書從警察職權行使、搜索、拘捕、偵訊、審判到執行，一一解答說明刑事案件進行的程序；讓法律從殿堂上走下，成為日常生活中，所有人都可以親近熟習的常識。
